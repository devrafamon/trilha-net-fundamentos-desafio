---

# 📚 Sobre o Projeto

Este desafio foi criado para praticar os **fundamentos da linguagem C# dentro do ecossistema .NET**.

Durante o desenvolvimento foram aplicados conceitos essenciais da programação, transformando conhecimento teórico em implementação prática.

Projetos desse tipo são comuns em trilhas de aprendizado porque ajudam a **consolidar lógica de programação, estruturação de código e organização de classes**, habilidades fundamentais para evoluir para aplicações mais complexas no ecossistema .NET. ([DIO][1])

---

# 🎯 Objetivo do Desafio

O principal objetivo do desafio é colocar em prática conceitos fundamentais como:

* lógica de programação
* manipulação de dados
* estrutura de classes
* organização de código
* interação com o usuário via console

Esse tipo de projeto funciona como um **exercício prático para consolidar as bases necessárias para desenvolvimento em .NET**.

---

# ⚙️ Funcionalidades Implementadas

Dependendo da execução do programa, o usuário pode interagir com diferentes funcionalidades através do console.

Entre as principais operações implementadas estão:

* interação com o usuário via terminal
* manipulação de estruturas de dados
* execução de operações baseadas em lógica de programação
* organização do código utilizando classes e métodos

O projeto demonstra como estruturar uma **aplicação simples em C# utilizando .NET**.

# 🧠 Conceitos Aplicados

Durante o desenvolvimento do projeto foram utilizados conceitos importantes como:

### 📦 Programação Orientada a Objetos

Utilização de classes e métodos para organizar o código de forma estruturada.

### 🔄 Estruturas de Controle

Uso de:

* condicionais (`if`, `switch`)
* estruturas de repetição (`while`, `for`)

### 📋 Manipulação de Coleções

Uso de estruturas como listas para armazenar e manipular dados.

### 🖥️ Aplicações Console

Interação com o usuário através de entrada e saída de dados no terminal.
